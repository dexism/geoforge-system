# GeoForge Project Structure

/ (プロジェクトルート)
|-- index.html              (エントリポイント)
|-- style.css               (グローバルスタイル)
|-- project.md              (プロジェクト構造図)
|-- AI_RULES.md             (AIアシスタント用ルール)
|-- src/                    (ソースコードディレクトリ)
    |-- main.js             (アプリケーションのエントリポイント・メインループ)
    |-- config.js           (設定・パラメータ定義)
    |-- utils.js            (汎用ユーティリティ関数)
    |-- ui.js               (UI操作・イベントハンドリング)
    |-- WorldMap.js         (ワールドマップ全体のデータ管理・状態管理)
    |-- MapView.js          (マップの描画・レンダリングロジック)
    |-- BlockLoader.js      (マップデータのブロック分割読み込み管理)
    |-- BlockUtils.js       (ブロック座標系の計算ロジック)
    |-- MapSplitter.js      (マップ分割ユーティリティ)
    |-- continentGenerator.js (地形・大陸生成ロジック)
    |-- civilizationGenerator.js (文明・人口分布生成ロジック)
    |-- economySimulator.js (経済・交易シミュレーション)
    |-- roadGenerator.js    (道路・交易路の生成ロジック)
    |-- infoWindow.js       (情報ウィンドウの表示・管理)
